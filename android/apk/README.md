# Production APK Output

This folder contains production-ready Android APK files generated by the build process.

## Current APK

- **File**: `investment-app-release.apk`
- **Type**: Signed, production-ready APK
- **Features**:
  - Code obfuscation (ProGuard)
  - Resource shrinking
  - Release signing
  - Optimized for Google Play Store

## Installation

Install on Android device via ADB:

```bash
adb install android\apk\investment-app-release.apk
```

Or manually transfer to device and install.

## Build Command

Generate a new production APK:

```bash
npm run android:release
```

## Notes

- APK files in this folder are `.gitignore`d to keep them out of version control
- Each build overwrites the previous APK
- For versioning, upload to Play Store or rename manually with version number
